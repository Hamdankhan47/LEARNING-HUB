HTML List Types

This page demonstrates the different types of HTML lists and explains how to use their attributes effectively.

1. Unordered List (<ul>)

An unordered list is used when the order of items doesnâ€™t matter. Each item is wrapped inside a <li> (list item) tag. By default, items are displayed with bullet points.

2. Ordered List with Attributes

An ordered list (<ol>) is used when the order of items is important. Items are numbered by default. You can customize the list using these attributes:

  • type: Changes numbering style. Common values include:
    • 1: Numbers (default)
    • A: Uppercase letters
    • a: Lowercase letters
    • I: Uppercase Roman numerals
    • i: Lowercase Roman numerals
  • start: Sets the starting value.
  • reversed: Displays the list in reverse order.

Example:

  1. First
  2. Second
  3. Third

3. Ordered List with Custom Value

You can manually override the numbering of individual list items using the value attribute inside <li>. This is useful when continuing lists or creating custom steps.

  1. Tenth item
  2. Next item

4. Description List (<dl>)

A description list is used to define terms and their corresponding definitions. It uses:

  • <dl>: The container (Description List)
  • <dt>: Term name (Definition Term)
  • <dd>: Description or explanation (Definition Description)
